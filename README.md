# 🍽 Food Journal App

A modern and visually appealing food tracking web application built using Flask, JSON, and a clean glassmorphism UI. This app helps users log their daily meals, monitor calorie intake, and maintain a healthy lifestyle in a simple and interactive way.

---

## ✨ Features

* 🔐 User Authentication (Login & Register)
* 🍱 Add meals with food name, calories, and type
* 📸 Upload meal images for better tracking
* 🗑 Delete meals easily
* 📊 Visual calorie tracking using charts
* 🎨 Modern UI with gradient + glassmorphism design
* 📱 Responsive layout (works on different screen sizes)
* 🧠 Lightweight data storage using JSON (no database required)

---

## 🛠 Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **Data Storage:** JSON file (no database)
* **Charts:** Chart.js
* **Authentication:** Flask-Login

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/dronamadhuri/food-journal-app.git
```

2. Navigate to the project folder:

```
cd food-journal-app
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the application:

```
python app.py
```

5. Open your browser and go to:

```
http://127.0.0.1:5000
```
